# Sessions Folder

Optional per-task notes for long-running work.

Naming recommendation:
- `YYYY-MM-DD-<task-id>.md`

Create from:
- `../templates/session-task.template.md`

Purpose:
- Keep each session self-contained.
- Allow a new AI session to continue from Git + session note without hidden context.
- Preserve TDD evidence (RED/GREEN/REFACTOR) and rule execution trace.
